# Meshy 3D Printing Academy

交互式 3D 打印学习平台，引导新用户从零基础成长为创作者。

## 功能

- 5 个学习阶段，循序渐进
- 完成教程获得积分和徽章
- Meshy 操作熟练值系统
- 进度自动保存

## 本地开发

直接用浏览器打开 `index.html` 或使用本地服务器：

```bash
npx serve .
```

## 部署

推送到 GitHub 后自动部署到 GitHub Pages。

## 技术栈

- 纯静态 HTML/CSS/JavaScript
- localStorage 存储
- GitHub Pages 托管
